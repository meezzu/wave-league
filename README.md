# waveleague

## Installation

1. Install yarn

```sh
$ npm install --global yarn
```

2. Install the dependencies

```sh
$ yarn
```

3. Install Typescript

```sh
$ npm install --global typescript
```

4. Start Typescript Compiler

```sh
$ tsc -w -p ./tsconfig.json
```

5. Open another shell and start the server
```sh
$ yarn start:dev
```
